Install Hexo In Ubuntu

#Ubuntu下部署Hexo

（1）安装npm

（2）安装nodejs

（3）安装hexo

（4）创建blog目录（进入到指定目录创建）

（5）安装依赖插件

（6）编辑你的blog

（略）

（7）转化为静态页面

（8）启动hexo服务器

（9）查看blog效果(本地查看)

（10）部署到github

也可以部署到不同类型的服务器
https://hexo.io/docs/deployment.html

编辑hexo下的_config.yml配置文件（注意空格）

这里https://github.com/BrightWind/TestBlog.git
实际上应该是你github主页地址，如
https://github.com/BrightWind/BrightWind.github.io

（11）查看blog效果(github查看)

https://brightwind.github.io/helloworld.html

（12）一些基本的操作命令

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
从hexo server启动服务后，在helloworld页面可以查看详细用法
其他可参考以下博主文章
http://ibruce.info/2013/11/22/hexo-your-blog/
http://www.tuicool.com/articles/YneiAzy
