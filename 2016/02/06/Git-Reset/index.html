Git Reset

git reset可用于不同的Commit创建分支

修改head位置

–soft

修改head位置并重置index

–mixed（默认）

修改head位置，重置index和工作副本

–hard

重置到某个commit,但是本地修改过的文件不会被重置

–keep

切换到前1个commit

git reset –soft HEAD^

前换到前3个commit,该命令会修改当前分支head，通常先创建并检出分支后再重置head

$ git reset –hard HEAD~3

重置某个文件的索引

git reset – frotz.c

切换到前三个分支，但修改过的文件不会被重置

git reset –keep HEAD~3
