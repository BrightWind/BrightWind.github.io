Git Branch

查看本地分支

$ git branch

查看远程分支

$ git branch -r
/master

查看所有分支

$ git branch -a
master
remotes/origin/master

若要查看各个分支最后一个提交对象的信息

$ git branch -v
iss53 93b412c fix javascript issue

  • master 7a98805 Merge branch ‘iss53’
    testing 782fd34 add scott to the author list in the readmes

查看哪些分支已经合入当前分支

$ git branch –merged
iss53

  • master

删除分支

#-D 强制执行
$git branch -d master

查看尚未合并的工作：

$ git branch –no-merged
testing

创建分支

git branch test_branch
